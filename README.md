This Smartphones Sales Dashboard, built using Power BI, SQL, and MySQL, provides a real-time view of smartphone sales performance across brands, models, cities, and payment methods.
It highlights key metrics such as total sales, transactions, quantity sold, average sales, and customer ratings, helping stakeholders quickly identify trends and top-performing products.
Interactive filters allow users to drill down by month, brand, and mobile model, enabling easy comparison and data-driven decision-making
